# 🎮Custom-RPC-generator

## 📌 Overview

This tool allows you to create a **custom Discord Rich Presence (RPC)** using Python.

With it, you can set your own Discord status including:

* Title (details)
* Description (state)
* Activity type (Listening, Playing, Watching, Competing)

The RPC updates your Discord profile in real-time and runs in the background.

---

## ⚙️ Features

* 🎨 Custom RPC title & description
* 🔄 Multiple activity types:

  * Listening
  * Playing
  * Watching
  * Competing
* 🧵 Runs in a background thread
* 🛑 Start and stop RPC anytime
* 🖥️ Works on Windows

---

## 📦 Installation

### 1. Make sure to install python

### 2. Install requirements

install the file "requirements.txt"

then run this command in cmd or powershell 

```bash
pip install -r requirements.txt
```

---

## 🚀 Usage

Start the program:

```bash
python main.py
```

### Menu options:

```
01 -> Set custom RPC
02 -> Exit
```

### Setup process:

1. Enter your RPC title
2. Enter your RPC description
3. Choose activity type:

   * 1 → Listening
   * 2 → Playing
   * 3 → Watching
   * 4 → Competing

After that, your custom RPC will appear on your Discord profile.

---

## 🛑 Stopping the RPC

While the RPC is running, you can:

* Stop it via the menu
* Or close the program

---

## ⚠️ Requirements

* Python 3.x
* Discord must be running

---

## ❗ Notes

* The tool uses a predefined Discord Application ID
* The program must stay open for the RPC to remain active
* If Discord is closed, the RPC will stop
